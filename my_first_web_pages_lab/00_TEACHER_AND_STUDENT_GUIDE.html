My First Web Page Lab

HTML = content. CSS = design. Later: JavaScript = action.

Teacher plan

  1. Students open 01_change_text.html on your website.
  2. They press Ctrl + S and save the page to Downloads.
  3. They open the saved file with Notepad.
  4. They find STUDENT EDIT AREA.
  5. They edit, save with Ctrl + S, and refresh browser with F5.

Student words

Browser = see the web page.

Notepad = edit the code.

Ctrl + S = save.

F5 = refresh.

Files

FileGoalStudent task
01_change_text.html HTML content Change “Hello everyone” to “Hello [name]”.
02_change_colors.html CSS design Change background color and text color.
03_change_background_image.html CSS image background Put an image in the same folder and change the file name in CSS.

Important warning

For the image task, the image file and the HTML file must be in the same folder.

Example: if the real image is cat.png, the code must say url("cat.png").
